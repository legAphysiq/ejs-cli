### EJS on the command line

##### Usage: 
	ejs-cli filename [-d] [-o output] [-p options]

```
Options:
  -d, --data     Data
  -o, --output   Output
  -p, --options  Options
  -h, --help     Show help                                             [boolean]
```

##### Example:
ejs-cli ./foo/bar.ejs -d ./data.json -o ./foo/bar.html -p ./options.json
