### EJS on the command line

##### Usage: 
	ejs-cli filename [-o output] [-d data] [-p options]

```
Options:
  -o, --output   Output
  -d, --data     Data
  -p, --options  Options
  -h, --help     Show help                                             [boolean]
```

##### Example:
	ejs-cli ./foo/bar.ejs -o ./foo/bar.html -d data.json -p options.json
